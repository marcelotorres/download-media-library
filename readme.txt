Download the files from the Media Library in ZIP format.

== Description ==

Download the files from the Media Library in ZIP format, <strong>organized by post type > post name > media type > file extension</strong>.

= Contribute =

* Github: https://github.com/marcelotorres/download-media-library/
* Donate: https://www.paypal.com/cgi-bin/webscr?cmd=_s-xclick&hosted_button_id=G85Z9XFXWWHCY

== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Media -> Download Media Library
